# DeWebSDK

#### 介绍
DeWeb : Delphi开发者的Web解决方案！  
DeWeb是一个可以直接将Delphi程序快速转换为网页应用的工具！  
使用DeWeb, 开发者不需要学习HTML、JavaScript、Java、PHP、ASP、C#等新知识，用Delphi搞定一切。   
DeWeb开发的网页支持所有客户端，包括手机、平板等。
