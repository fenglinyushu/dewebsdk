为什么研发DeWeb？
___
#### 情迷Delphi！梦断Delphi!
二十多年来，编程语言Delphi 以对 Windows 应用程序的快速可视化编程而闻名于世。  
尤其是轻松开发桌面数据库应用程序和快捷简单的将应用程序部署到客户系统，  
使得 Delphi 与其他编程语言相比更受程序员欢迎。

我们的开发团队一直使用Delphi进行开发，从Delphi 3到后面主力使用的Delphi 7,  
完成了一个又一个成功的系统，也缔造了一个又一个辉煌！  
其中一个项目做到全国200多家，在同行业内占据绝对垄断地位。  

随着时代的发展，Web借互联网大潮兴起！  
为了改成B/S系统，我们尝试过很多方案，  
如：IntraWeb、SilverLight、CBX等都没有很好解决。  
包括我们后来研发的WebXone技术，也因为浏览器全面禁止ActiveX而未果。  
&nbsp;  
更新Delphi系统到B/S系统，不管采用IntraWeb，还是后面尝试的PHP，甚至Java，  
都需要更新大量的web知识，   
包括HTML、JavaScript、CSS、AJAX等，  
这样基本需要全部更换技术团队，而且原来大量的代码和资源将无法使用，  
所有代码需要重新编写，模块需要重新测试。  
为此，必须有一个基于Delphi的Web解决方案！   
&nbsp;   
#### DeWeb 因应而生！
DeWeb 意即：Delphi - Web  
从2015年开始研发，最终发展为一个独立的开发框架！   
支持电脑、手机和平板！  
可开发网页、公众号和小程序！
